# Employee Task Management System

## 📌 Project Overview

Employee Task Management System is a web-based application developed using ASP.NET Core MVC and SQL Server.  
The system helps organizations manage employees, projects, and tasks efficiently through a centralized dashboard.

This project provides task assignment, employee management, project tracking, authentication, analytics, and reporting features with a modern responsive UI.

---

# 🚀 Features

## 🔐 Authentication & Security
- Login Authentication
- Role-Based Login System
- Admin & Employee Dashboard
- Session Management

## 👨‍💼 Employee Management

- Delete Employee
- Search Employee
- Profile Image Upload

## 📋 Task Management
- Create Tasks
- Assign Tasks

- Live Task Updates

## 📁 Project Management
